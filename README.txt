RushHour / Ane Rouge.

Games made in the aim of the UE "ED & PP1"
Project made by Valentin Gaillard, Juline Pilleux, Ali Cherifi.

Prerequisites : CMake

How to install (any machine) :

	- Open a terminal in the folder you extract  the archive.
	- type "cmake ./" (without the quotes)
	- type "make install"
	- Play !
